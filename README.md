# MyLibrary

Home of the **`knowledge-library`** agent skill — abstracted from the MyBooks project.

It turns a growing pile of self-contained, article-sized text units (Zhihu saves, video/podcast
summaries, web clippings, research reports, book-chapter summaries) into a **browsable,
topic-linked Obsidian library**: a config-driven pipeline that designs a library-model taxonomy,
labels every item, builds a topic knowledge graph, and materializes a clean vault — plus an
autonomous steady-state mode to pull and file new items on a schedule.
